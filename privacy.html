ระบบขอให้ดำเนินการกิจกรรมนักศึกษา (ระบบ Check-In / Passcode)

กองพัฒนานักศึกษา มหาวิทยาลัยราชภัฏพระนครศรีอยุธยา ให้ความสำคัญกับการคุ้มครองข้อมูลส่วนบุคคลของผู้ใช้งานระบบ โดยนโยบายฉบับนี้แจ้งให้ทราบถึงการเก็บรวบรวม ใช้ จัดเก็บ และดูแลข้อมูลส่วนบุคคลที่เกี่ยวข้องกับการใช้งานระบบ

1. ข้อมูลที่ระบบจัดเก็บ

  • ชื่อ–สกุล หน่วยงาน และเบอร์โทรศัพท์
  • LINE User ID สำหรับส่งข้อความแจ้งเตือนผ่าน LINE Official Account
  • ข้อมูลคำขอกิจกรรม เช่น ชื่อกิจกรรม วันที่ เวลา สถานที่ จำนวนผู้เข้าร่วม ประเภทระบบ และระดับกิจกรรม
  • ลิงก์เอกสารประกอบการขอ เช่น กำหนดการกิจกรรม รายชื่อผู้เข้าร่วม และไฟล์ที่เกี่ยวข้อง
  • ข้อมูลสถานะการดำเนินการและหมายเหตุจากเจ้าหน้าที่

2. วัตถุประสงค์ในการใช้ข้อมูล

  • เพื่อรับคำขอให้ดำเนินการระบบกิจกรรมนักศึกษา
  • เพื่อบริหารข้อมูลกิจกรรมในระบบ Check-In และ Passcode
  • เพื่อให้เจ้าหน้าที่ตรวจสอบ อัปเดตสถานะ และดำเนินการตามคำขอ
  • เพื่อแจ้งสถานะการดำเนินการผ่าน Web App และ LINE Official Account
  • เพื่อจัดเก็บประวัติคำขอและตรวจสอบย้อนหลังภายในหน่วยงาน

3. การเปิดเผยข้อมูล

ข้อมูลที่จัดเก็บในระบบจะใช้เพื่อการดำเนินงานภายในมหาวิทยาลัยเท่านั้น และจะไม่เปิดเผยต่อบุคคลภายนอก เว้นแต่จำเป็นต่อภารกิจของมหาวิทยาลัยหรือเป็นไปตามที่กฎหมายกำหนด

4. การจัดเก็บและความปลอดภัยของข้อมูล

ข้อมูลจะถูกจัดเก็บในระบบที่กองพัฒนานักศึกษากำหนด เช่น Google Sheet และระบบที่เกี่ยวข้อง โดยจำกัดสิทธิ์การเข้าถึงเฉพาะผู้ที่เกี่ยวข้องกับการดำเนินงานเท่านั้น

5. สิทธิของเจ้าของข้อมูล

ผู้ใช้งานสามารถขอแก้ไขข้อมูลของตนเองได้ผ่านระบบ หรือแจ้งเจ้าหน้าที่กองพัฒนานักศึกษา หากต้องการตรวจสอบ ปรับปรุง หรือแก้ไขข้อมูลที่ไม่ถูกต้อง

6. การแจ้งเตือนผ่าน LINE

ระบบอาจส่งข้อความแจ้งเตือนสถานะคำขอกิจกรรมไปยัง LINE Official Account ของผู้ใช้งาน โดยอ้างอิงจาก LINE User ID ที่ผู้ใช้งานให้ไว้หรือผูกไว้กับระบบ

หากไม่ประสงค์รับการแจ้งเตือนผ่าน LINE ผู้ใช้งานสามารถไม่กรอก LINE User ID หรือแจ้งเจ้าหน้าที่เพื่อปรับปรุงข้อมูลได้
